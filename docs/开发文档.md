# 开发文档（demo版，实际开发需完善）

**架构：** 前端vue3+ts+element-plus，后端cf workers+ts
**定位：** 类似chatBOX的大模型API配置聊天程序

**实体类：** 
- 智能体（id，名称、角色描述、第一条消息）
- 聊天记录（id，智能体id，消息数组）
- 消息（id，聊天记录id，角色，内容，自定义提示词id，meta数据）
- 自定义提示词（id，名称，内容）
- 大模型配置（id，模型名称，模型api地址，模型api密钥，模型参数）
- 组装请求体（id，大模型配置id，聊天记录id，消息id数组）

**方法**
- 解析模型响应数据

**逻辑特点：** 前端直接输入符合语法的模型参数字符串，后端直接把这个字符串解析成json数组作为模型参数

**前端特点：** 
- 质感设计圆角风格UI，主题色为粉+白
- 消息内容以markdown格式化渲染
- 全局背景可更换，支持图片和颜色。
- UI交互布局说明：
  - 左侧为可折叠的智能体列表，选中某智能体后，继续向右拓展显示该智能体的聊天记录列表，点击聊天记录后在聊天界面中显示该聊天记录的详细内容。右上角可点击设置按钮切换到智能体设定页面，包含智能体名称、角色描述、第一条消息。该折叠列表是浮在聊天界面上的，其UI尺寸变化不会影响聊天界面布局。
  - 中间为聊天界面，呈现聊天记录和输入界面。透明度可调。
  - 聊天界面右上角可点击切换到设置页面，包含大模型配置、自定义提示词。
